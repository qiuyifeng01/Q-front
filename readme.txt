\目录说明
│  blank				空白栏目(每次我们都拿空白栏目去创建，这样比较干净！)
│  public				公共文件(可以复用的如jq库，reset.css)
│  lab   				实验目录(用于测试的调试目录)
│  temp   				一些常用html+css+js结构模块(方便以后复制复用)
│  pages   				一些完整的页面(具有借鉴性的页面)
│  demo   				一些说明的例子可以放在这个目录
│  origin   				一些插件的原始文件可以放在这里
│  plus   				经过自己改造的插件
